다양한 방법으로 요소를 숨겨봅시다.

  • hidden
    hidden은 input 요소의 특성이다.
  • display:none
    (요소가 사라짐: 없는 것처럼 취급 됨)
  • opacity(투명도): 0
    (보이지만 않음)
  • width:0 요소의 크기를 0으로 만들어도 컨텐츠는 없어지지 않음.
    컨텐츠
  • width:0
    컨텐츠
    넘쳐흐르는 컨텐츠를 hidden으로 한다
  • box-sizing : border-box 로 해도 크기가 무관하게 border가 생성 (무조건 border-width = 0 으로 지정, border:none)
  • visibility: hidden
    요소는 존재하지만 보이지 않게 한다
  • transform 을 이용해 제거 실제로 줄었지만 공간은 남음
