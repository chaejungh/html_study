요소 속성 정의 법

요소의 속성은 노드개체의 상태를 바꾼다.

노드개체 : 요소가 브라우저에 출력된 상태(객체, 이벤트 리스너가 정의됨)

input 요소의 type 속성

type, name, value는 input 요소만 사용하는 속성

  • type의 기본값은 text
  • :checked 속성자체에 true값을 갖는다.(없으면 false)
  • 시집읽기 , 웹툰읽기 , 만화읽기 ,
  • 1억받기 10억받기 50%

공통 속성

id, class, data-*, style

  • id : html 문서 내에서 유일한 요소를 지정
  • class : 복수의 이름을 지정
  • data-* : 사용자 지정 속성
  • style : 요소에 그래픽을 지정
  • title = 툴팁제공
  • hidden : 보이지 않게 합니다. (hidden 이 있으면 true)

리소스를 참조하는 속성

href(Hypertext REFerence), src(reSouRCe)

  • 상대주소로 이동
  • 절대주소로 다른 서버로 이동
