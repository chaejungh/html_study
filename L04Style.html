CSS 스타일

브라우저는 html을 출력할때 CSS를 기반으로 적용합니다.

스타일의 종류

  • color : text 색 자식요소는 부모요소의 스타일을 그대로 적용(CASCADING)
  • background-color : 배경색 #FF6464 (00 ~ FF(0~255)red 255 green 100 blue 100) : 같은 속성일때 자식의 우선순위가 높다. (override)
  • asdd
  • font-size : 폰트크기 -weight : 폰트 굵기
  • asdad
  • text-align: [ left, center, right]
  • 여백 외부(margin), 내부(padding)
